Introduction

My Website

This is my journal for my summer internship in Nanjing at Sheninside Company. Journal entries generally take me about 40 minutes to write, and more time to add new code and new functions to the website. My goal is to make this website visually appealing and as complex as possible, evolving as my knowledge of programming grows, yet remain easy to understand for people without basic programming knowledge. My journal entries should be easily accessible and website navigation should be convenient.

Purpose

I am using this website as a practice for the programming skills I have learned through online self-studying as well as a place to record my experiences for documentation and share them with others. It's also something to keep me busy during the workday since writing journal entries is very time consuming and I would not have time to write in such detail after work.

Functionality

Each journal entry has buttons at the bottom of the page which will link to the previous and next pages of the journal. To skip to a specific journal entry, please use the Select Entry tab in the navigation bar at the top of the page or the navigation bar at the bottom of the page. The Highlights tab contains a much briefer summary listing the highlights of my time in Nanjing.

Contents

Journal entries begin with the entry number and date and time of completion. Each entry includes several subtitles indicating the major event which the following paragraphs are about. The subject of each journal entry is the notable events which occured between the completion times of the current and previous entries. If you are looking for a specific event, find the event in the Highlights page and click the corresponding box to jump to the entry.

The various components of the website have been created using resources and code as follows:

  • Website title: HTML
  • Navigation bar and styling: CSS
  • Background Color: CSS
  • Home Page: Bootstrap
  • Text: HTML
  • Text styling: CSS
  • Navigation buttons: JS
  • Local website host: XAMPP (Apache)
  • Online website host: GitHub
  • Page navigation: Bootstrap
